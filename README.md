# 🌗 Dark / Light Mode Toggle

## 📌 Description

A simple web application that allows users to switch between **dark mode** and **light mode**.
This project demonstrates DOM manipulation and dynamic UI updates using JavaScript.

---

## 🚀 Features

* 🌗 Toggle between light and dark mode
* ⚡ Instant UI update
* 💾 Save user preference using localStorage

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
